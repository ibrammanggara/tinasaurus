# **Tutorial Dasar Perintah Linux**

Selamat datang di tutorial Dasar Perintah Terminal Linux! Dalam tutorial ini, kita akan belajar beberapa perintah dasar yang sering digunakan di terminal Linux. Menguasai perintah-perintah ini akan membantu kamu berinteraksi dengan sistem Linux dengan lebih efisien.

### Mengapa Menggunakan Terminal?

Terminal adalah alat yang sangat kuat untuk berinteraksi dengan sistem operasi. Melalui terminal, kita dapat menjalankan perintah untuk mengelola file, mengatur hak akses, menginstal perangkat lunak, dan banyak lagi.

### Perintah Dasar yang Perlu Diketahui

Berikut adalah beberapa perintah dasar yang akan kita pelajari:

```
sudo
```

Perintah \`sudo\` digunakan untuk menjalankan perintah dengan hak akses superuser (root).

```
pwd
```

Menampilkan direktori tempat kamu berada saat ini.

```
ls
```

Menampilkan daftar file dan folder dalam direktori saat ini.

```
cd
```

Pindah antar direktori.

```
mkdir
```

Membuat folder baru.

```
rm
```

Menghapus file atau folder.

```
cp
```

Menyalin file atau folder ke lokasi lain.

```
mv
```

Memindahkan atau mengganti nama file/folder.

```
cat
```

Menampilkan isi file.

### Kesimpulan

Memahami perintah dasar di terminal Linux sangat penting untuk berinteraksi dan mengelola sistem operasi dengan lebih baik. Teruslah berlatih dan eksplorasi perintah-perintah lainnya untuk memperdalam pemahamanmu. Semoga tutorial ini membantu kamu memulai perjalananmu di dunia Linux!
