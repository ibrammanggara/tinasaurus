\## Menggunakan pgAdmin dengan Docker

Sekarang saatnya untuk menjalankan pgAdmin menggunakan Docker. pgAdmin adalah alat manajemen database yang kuat dan mudah digunakan. Mari kita mulai!

1\. Jalankan perintah ini untuk mengunduh dan menjalankan container pgAdmin dari Docker Hub:

          visibility: hidden !important;

        }

-v /tmp/servers.json:/pgadmin4/servers.json \\

-e 'PGADMIN\_DEFAULT\_EMAIL=studentcloud121\@gmail.com' \\

-e 'PGADMIN\_DEFAULT\_PASSWORD=yy123@#$' \\

-e 'PGADMIN\_ENABLE\_TLS=True' \\

2\. Sekarang, Anda dapat mengakses pgAdmin melalui browser web Anda dengan menavigasi ke \`https\://alamatip/domain\`. Gunakan alamat email dan kata sandi yang telah Anda tentukan untuk masuk.
