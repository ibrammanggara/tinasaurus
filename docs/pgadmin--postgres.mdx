\# Petualangan Docker: Menginstal dan Menggunakan pgAdmin dengan Mudah!

\## Selamat Datang di Petualangan Docker!

Selamat datang di petualangan instalasi dan penggunaan Docker! Anda siap untuk memulai perjalanan yang menarik menuju dunia kontainerisasi? Mari kita mulai!

\## Instalasi Docker

Langkah pertama adalah menginstal Docker di sistem Anda. Docker memungkinkan Anda untuk menjalankan aplikasi di lingkungan kontainer dengan mudah. Berikut langkah-langkahnya:

1\. Buka terminal favorit Anda dan jalankan perintah:

```
sudo apt update
```

```
sudo apt install apt-transport-https ca-certificates curl software-properties-common
```

2\. Setelah itu, tambahkan kunci GPG resmi Docker ke sistem Anda:

```
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
```

3\. Tambahkan repositori Dcker ke daftar sumber perangkat lunak: 

```
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
```

4\. Terakhir, instal Docker dengan perintah:

```
sudo apt install docker-ce
```

\## Menggunakan pgAdmin dengan Docker

Sekarang saatnya untuk menjalankan pgAdmin menggunakan Docker. pgAdmin adalah alat manajemen database yang kuat dan mudah digunakan. Mari kita mulai!

1\. Jalankan perintah ini untuk mengunduh dan menjalankan container pgAdmin dari Docker Hub:

```
docker pull dpage/pgadmin4
```

```
docker run - p 443: 443 \
-v / data: /var/lib / pgadmin \
-v / home / ubuntu / server.cert: /certs/server.cert \
-v / home / ubuntu / server.key: /certs/server.key \
-v / tmp / servers.json: /pgadmin4/servers.json \
-e 'PGADMIN_DEFAULT_EMAIL=studentcloud121@gmail.com' \
-e 'PGADMIN_DEFAULT_PASSWORD=yy123@#$' \
-e 'PGADMIN_ENABLE_TLS=True' \
-d dpage / pgadmin4
```

2\. Sekarang, Anda dapat mengakses pgAdmin melalui browser web Anda dengan menavigasi ke \`https\://localhost:443\`. Gunakan alamat email dan kata sandi yang telah Anda tentukan untuk masuk.

\## Selamat! Anda Telah Berhasil!

Selamat! Anda telah berhasil menginstal Docker dan menjalankan pgAdmin dengan menggunakan Docker. Kini Anda siap untuk menjelajahi dunia manajemen database dengan mudah dan cepat. Semoga Anda menikmati petualangan ini!
